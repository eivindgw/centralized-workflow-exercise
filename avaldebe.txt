
My favourite git alias

```bash
# history: pretty log one-liner
git config --global alias.history 'log --pretty=tformat:"%h %ad: %s" --date=short'

# last: history for last commit/HEAD
git config --global alias.last    '!git --no-pager history HEAD -1'

# who: like history but show commit author
git config --global alias.who     'log --pretty=tformat:"%h %ad: %an" --date=short'
```
